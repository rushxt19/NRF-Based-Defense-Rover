# NRF-BASED-DEFENCE-ROVER

An intelligent and remotely operated defence rover designed for surveillance, monitoring, and reconnaissance applications using NRF24L01 wireless communication. The rover is capable of real-time movement control, obstacle detection, and live environmental monitoring, making it suitable for defence and hazardous-area operations.


## Features
Wireless communication using NRF24L01 transceiver modules
Real-time remote control of rover movement
Obstacle detection using ultrasonic sensors
Camera module integration for surveillance and monitoring
Rugged rover chassis for rough terrain navigation
Low-power and long-range communication system
Expandable architecture for future AI and IoT integration


## Hardware Components
Arduino Nano
NRF24L01 Transceiver Modules
L298N Motor Driver
DC Motors with Wheels
Ultrasonic Sensor (HC-SR04)
ESP32-CAM / USB Camera Module
IR Sensors
Li-ion Battery Pack
Rover Chassis
💻 Software Used
Arduino IDE
Embedded C / C++
NRF24 Library
Serial Communication Protocols


## Working Principle
The transmitter module sends control commands wirelessly through the NRF24L01 communication module. The receiver side processes the received signals using the Arduino Nano and controls the rover motors through the motor driver module. Sensors continuously monitor the surroundings for obstacles and environmental conditions, while the camera module provides live surveillance capability.
